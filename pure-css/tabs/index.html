🔷 React

  • 作者：Meta（Facebook）
  • 初始发布：2013
  • 核心理念：声明式 UI + 虚拟 DOM + 函数组件 + Hooks
  • 特点：
    • 使用 JSX 编写组件
    • 通过 useState、useEffect 等 Hook 管理状态
    • 强大生态系统（如 Next.js）

🟢 Vue

  • 作者：尤雨溪（Evan You）
  • 初始发布：2014
  • 核心理念：响应式数据 + 模板语法 + 组件系统
  • 特点：
    • 易上手，文档完善
    • 支持 Options API 和 Composition API
    • 拥有官方全家桶（Vue Router、Pinia、Nuxt 等）

🟡 SolidJS

  • 作者：Ryan Carniato
  • 初始发布：2020
  • 核心理念：编译时优化 + 无虚拟 DOM + 细粒度响应式
  • 特点：
    • 语法类似 React（JSX + Signals）
    • 性能极高，接近原生 DOM 操作
    • 没有 Virtual DOM，运行时开销极低
